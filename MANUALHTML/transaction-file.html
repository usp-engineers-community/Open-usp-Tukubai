トランザクションファイル(5)

【名前】

トランザクションファイル : トランザクションの性質を満たすフィールド形式のファイル

【書式】

第1フィールド 第2フィールド 第3フィールド ...
...

【説明】

売上データなど随時データの追加などが実施されるタイプのデータはトランザクションと呼ばれている。トランザクションは短縮してトランと呼ばれることも多い。

ユニケージ開発手法では、トランザクションに該当するフィールド形式のファイルをトランザクションファイルと呼んでいる。

トランザクションファイルは短縮してトランと呼ばれることがあり、また、Tukubaiコマンドマニュアルではtranと表記される。トランザクションファイルとは対照的にデータ変更のタイミングがあらかじめ定めされており、キーフィールドが重複せずかつ整列されているようなフィールド形式のファイルはマスタファイルと呼ばれている。

【例1】

トランザクションファイルの例は次のとおり。

20110405 001 13513543
20110405 002 3403346
20110406 001 3543024
20110406 002 12412311
20110406 003 88797982
20110407 001 1354131
20110407 002 12343423
20110407 003 7657765
